Ensure you have Node.js installed on your machine.

Clone the repository and install the dependencies using
1. npm install
2. npm playwright install

Before running the tests, you must provide your own API key. The project is designed to pull credentials from a central utility file to keep the codebase clean.
1. Navigate to utils/config.js.
2. Locate the API_KEY property.
3. Replace the placeholder text with your actual Reqres API key.
